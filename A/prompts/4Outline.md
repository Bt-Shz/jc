# outline 
Using this thesis:
[paste thesis]

Build a philosophy essay outline for a 1400–1600 word paper.

Requirements:
- thesis should appear in the introduction
- each body paragraph must contribute directly to defending the thesis
- include at least one strong objection and a response
- avoid broad generalizations
- focus tightly on the most relevant aspects only
- include places where I should use examples
- include places where I should use course readings
- do not use many quotations
- make the structure logical, cohesive, and concise

Output format:
1) title options
2) final thesis
3) paragraph-by-paragraph outline
4) objection section
5) conclusion strategy
6) list of what evidence I still need
# map 
Turn my outline into an argument map.

For each paragraph, identify:
- main claim
- reason supporting it
- example/evidence needed
- transition logic
- possible weakness

Then tell me whether the overall paper is:
- too descriptive
- too broad
- under-defended
- missing objections
- missing definitions

Be critical.
